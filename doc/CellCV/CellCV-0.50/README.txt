REQUIREMENTS:

 - a Cell Processor architecture (i.e. Sony PlayStation3)
 - a linux distribution for PowerPC (i.e. YellowDog linux 5.0 for PS3)
 - the Cell SDK 2.0 and the correspondent packages available at the
   Barcelona Supercomputing Center website
 - the GNU build environment

BUILDING INSTRUCTIONS:

 - extact the CellCV-0.50.tgz file in the directory
   /opt/ibm/cell-sdk/prototype/src/samples/tutorial/euler/CellCV-0.50 
 - enter the directory CellCV-0.50 just created, type "make" and then
   type "./CellCV_spe"

CLEANING INSTRUCTIONS:
 - enter the directory
   /opt/ibm/cell-sdk/prototype/src/samples/tutorial/euler/CellCV-0.50
   and type "make clean"
